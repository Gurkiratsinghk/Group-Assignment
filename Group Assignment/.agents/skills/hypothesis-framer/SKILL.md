# Skill: hypothesis-framer
# Description: Frames statistical null and alternative hypotheses.

## Core Logic & Rules:
1. Parse the business scenario description.
2. Identify the parameter of interest (mean mu, proportion p, variance sigma^2).
3. Determine the claim or effect the manager wishes to prove (the alternative hypothesis Ha).
4. Frame the Null Hypothesis H0 (the status quo or baseline to challenge).
5. Classify the test as:
   - One-tailed (Upper-tailed: Ha: mu > mu0)
   - One-tailed (Lower-tailed: Ha: mu < mu0)
   - Two-tailed (Ha: mu != mu0)
